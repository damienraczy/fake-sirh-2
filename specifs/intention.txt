mon-intention

Nous allons établir les spécifications d'un « faux SIRH » qui simule le fonctionnement d'une entreprise.

Il n'y aura pas d'interface, seulement une base de données.

Pour créer les données, nous utiliserons des CLI Python avec des appels à Ollama Turbo (ollama.com), un outil récemment mis sur le marché.
L'entreprise sera construite à partir d'une description macroscopique.

L'objectif sera ensuite d'utiliser ces données dans un RAG pour aider les entreprises à résoudre des problématiques RH ou organisationnelles.

Le projet s'appelle « Fake-sirh-2 ».

Le but est de faire rapidement le CLI, puis de prototyper rapidement le RAG, pour un petit déjeuner qui montrera l'intérêt du RAG sur des problématiques précises type analyse ou préparation d'EAE, pour un individu ou pour une unité organisationnelle.

Tu as les spécifs de plusieurs LLM, et des propositions de tables pour l'organisation.
